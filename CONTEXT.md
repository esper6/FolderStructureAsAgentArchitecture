# Workspace Routing

Given what you want to do, here is where to go.

## Workflows

### Build a new workspace from a goal description
Takes a freeform goal and produces a complete MWP workspace.

**Entry point:** Fill in `setup/goal-intake.md`, then run the stages in order.

| Stage | Folder | Job |
|---|---|---|
| 1 | `stages/01_assess_design/` | Assess fit + propose workflow for review |
| 2 | `stages/02_build/` | Generate all workspace files from the approved design |
| 3 | `stages/03_validation/` | Validate the manifest and fix issues before writing to disk |

---

## Shared resources
- MWP conventions and file format rules: `_config/mwp-conventions.md`
- Skills catalog: `_config/skills-catalog.md`
- Goal intake template: `setup/goal-intake.md`
